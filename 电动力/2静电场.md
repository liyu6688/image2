# 静电场的标势及其微分方程

### 静电场的标势

+ 静电场就是静止电荷产生的场，不是变化的电场产生磁场吗，没有这个变化了，就是麦克斯韦里面那个电场的旋度是0，电场磁场分离

+ 静电场
  + 无旋场
  + 保守力场
  + 对电荷的做功和路径无关
  + 所以引入标势描述E
+ 电势的两个求法
  + 一个根据电场来
  + 一个根据电荷来
